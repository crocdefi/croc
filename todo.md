# CROC Protocol Landing Page - Development Plan

## Design Guidelines

### Design References
- **Uniswap.org**: Clean dark Web3 aesthetic
- **Aave.com**: Glassmorphism + neon glow
- **Style**: Dark Futuristic + AI + Meme + DeFi

### Color Palette
- Background: #0a0a0a (Deep Black)
- Card BG: rgba(15, 15, 25, 0.8) (Dark Glass)
- Primary Accent: #00ff88 (Neon Green - crocodile theme)
- Secondary: #7b2fff (Purple glow)
- Tertiary: #00b4ff (Blue glow)
- Text Primary: #ffffff
- Text Secondary: #a0a0b0

### Typography
- Headings: Orbitron (futuristic display font)
- Body: Inter (clean sans-serif)

### Key Component Styles
- Cards: Glassmorphism with backdrop-blur, neon border glow
- Buttons: Neon green gradient, glow on hover
- Sections: Full-width with subtle separators
