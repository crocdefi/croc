# CROC Protocol Landing Page - Development Plan

## Design Guidelines

### Design References
- **Uniswap.org**: Clean dark DeFi aesthetic
- **Polygon.technology**: Purple/blue glow effects
- **Style**: Dark Futuristic + AI + Meme + DeFi

### Color Palette
- Primary BG: #0a0a0a (Deep Black)
- Secondary BG: #111111 (Dark Gray)
- Card BG: rgba(17, 17, 17, 0.7) (Glassmorphism)
- Accent Green: #00ff88 (Neon Green - crocodile theme)
- Accent Purple: #8b5cf6 (Purple glow)
- Accent Blue: #3b82f6 (Blue glow)
- Text Primary: #ffffff
- Text Secondary: #a0a0a0

### Typography
- Headings: 'Orbitron', sans-serif (futuristic)
- Body: 'Inter', sans-serif (clean readability)

### Key Component Styles
- Cards: Glassmorphism with backdrop-blur, border glow
- Buttons: Neon green gradient, glow on hover
- Sections: Alternating subtle backgrounds with glow accents
